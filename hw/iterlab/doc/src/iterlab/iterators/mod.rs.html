3//! This module contains 4 iterators that you will implement!
4//!
5//! Implement this module _before_ the [`hofs`](self) module.
6//!
7//! You don't _have_ to implement these in the order that we state, but we would recommend
8//! following our suggestions.
9//!
10//!
11//! # Fibonacci
12//!
13//! Implement the [`Fibonacci`](fibonacci::Fibonacci) sequence as an iterator!
14//!
15//! This is mainly a warmup for the next 3 parts.
16//!
17//!
18//! # Cycle
19//!
20//! Implement an iterator that takes as input another iterator, and instead of exhausting the input
21//! iterator when it reaches its last element, cycle back to the first element.
22//!
23//! You'll need to figure out what information we need to keep track of to implement this iterator,
24//! and store it in [`Cycle`](cycle::Cycle).
25//!
26//! Don't worry too much about performance here, just make sure it works!
27//!
28//!
29//! # Interleave
30//!
31//! Given two input iterators, create an iterator that interleaves the outputs of the two iterators.
32//! Note that we require the two input iterators to have the same associated `Item` type.
33//!
34//! Similarly to [`Cycle`](cycle::Cycle), you will need to figure out what to store in the struct
35//! [`Interleave`](interleave::Interleave).
36//!
37//!
38//! # Double
39//!
40//! Given a single input iterator, double every single output!
41//!
42//! Hint: What is the _easiest_ way to implement this using code you have already written?
43
