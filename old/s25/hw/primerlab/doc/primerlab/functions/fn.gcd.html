Function gcd

Source 
pub fn gcd(x: usize, y: usize) -> usize
Expand description

Returns the Greatest Common Divisor (gcd) of two numbers x and y.

§Example

use primerlab::functions::gcd;

let first_num = 60;
let second_num = 24;

assert_eq!(gcd(first_num, second_num), 12);

§Note

You must use Euclid’s algorithm to solve this problem. Please feel free to use the World Wide Web to learn more (this applies for the entire course).

§Restrictions

Please do not use the “return” keyword. Please write this function using recursion.
