fn find_next(&self, s: &str) -> Option<(usize, usize)>

Finds the next occurrence of the pattern in the given string.

Returns Some((start, end)) if the pattern is found, where start is starting index of the pattern and end is the index of the end of the pattern in the string. Returns None if the pattern is not found.
